# 新一代世界级爆款小游戏 · 最终研究结论

一份关于"世界级爆款小游戏"的定稿研究结论，以单文件可交互看板形式呈现。

## 核心结论

- **最火的世界级爆款小游戏（按全球装机/触达）是 Block Blast!（方块爆破）**：累计装机 5.5 亿次、2025 年 3–5 月连续三个月 Sensor Tower 全球第一；是纯轻量解谜品类里唯一登顶全球的小游戏。
- 若以"同时在线纪录"为尺，最火的是 **Grow a Garden（种植花园）**——2200 万同时在线世界纪录，验证"离线成长 + UGC"结构。
- "世界级爆款小游戏"是一组**可验证、可复用、跨文化**的产品结构，而非模糊口号：轻钩子 → 中承接 → 重变现；不依赖语言的动作核心；离线成长 + 每日回流钩子；合作优于竞争的留存；UGC 是内容容器核心。
